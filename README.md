# SUSHI-BIRTHDAY
Twitterの誕生日風船を寿司に変えるプラグイン

## 概要
Twitterで誕生日のユーザーのプロフィールを見ると風船が飛んでゆく。。  
この風船を🍣に変えるためのプラグイン。

## 機能
・誕生日のユーザーのプロフィール画面に飛んでくる風船を🍣に変える  
・ついでに、プロフィールの背景画像も寿司にする。
  (あくまで変更できるのは自分がみる画面だけで、他のユーザがみる画面は通常の風船のままです)

## 使い方
1. ChromeWebストアからプラグイン"[SUSHI-BIRTHDAY](https://chrome.google.com/webstore/detail/eabkdlkmjjidmhbbfooicbdepojhjman/publish-delayed?hl=ja)"をインストールする。
2. 誕生日のアカウントのプロフィール画面に行く
3. 🍣が浮かんできてハッピーな気分になれる
